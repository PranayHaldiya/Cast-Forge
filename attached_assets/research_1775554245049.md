## ElevenLabs API Capabilities (Confirmed)

### Voice Design API
- `POST /v1/text-to-voice/create-previews` → generates up to 3 voice previews from text description
- `POST /v1/text-to-voice/create` → saves a preview to voice library
- Model: `eleven_multilingual_ttv_v2` or `eleven_ttv_v3`
- Parameters: voice_description (20-1000 chars), text (100-1000 chars), guidance_scale, quality, loudness, seed

### Text-to-Dialogue API
- `POST /v1/text-to-dialogue` → renders multi-speaker dialogue as single audio
- `POST /v1/text-to-dialogue/with-timestamps` → with character-level timing
- Input: array of {text, voice_id} pairs
- Supports emotion tags: [laughing], [whispering], [excited], [sad], [groaning]
- Supports audio events: [leaves rustling], [applause], [footsteps]
- Supports interruptions (—) and trailing off (...)
- Model: eleven_v3 only
- Unlimited speakers per dialogue

### Sound Effects API
- `POST /v1/sound-generation`
- Parameters: text (description), duration_seconds (0.5-30s), prompt_influence (0-1), loop (boolean)
- Model: eleven_text_to_sound_v2
- 40 credits/second when duration specified

### Music Generation API
- `POST /v1/music/stream` → streams composed song
- `POST /v1/music/plan` → creates composition plan (free)
- Model: music_v1
- Duration: 3,000-600,000 ms (3s to 10min)
- Supports prompts describing genre, mood, instrumentation

### TTS Flash v2.5
- `POST /v1/text-to-speech/{voice_id}/stream`
- Model: eleven_flash_v2_5
- Latency: ~75ms
- 32 languages, 40,000 char limit
- 50% lower cost per character

## Market Validation: NotebookLM Audio Overviews

### What NotebookLM Does
- Converts documents into 2-host podcast discussions
- Formats: Deep Dive (10-30 min), Brief (<2 min), Critique, Debate
- Fixed two AI hosts (no customization)
- Interactive Mode: join and ask questions mid-conversation
- Generation time: 2-5 minutes
- Massive adoption — millions of audio overviews generated

### NotebookLM's Limitations (Our Opportunity)
- Documents only input (no free-form topics)
- Zero voice customization (same two hosts every time)
- No sound effects or music
- No shareable clips / social-friendly output
- Locked in Google ecosystem
- No format variety beyond 4 options
- No emotional delivery control

### CastForge Differentiators
- Any topic as input (not just documents)
- Custom host voices via Voice Design
- 6 show formats with distinct tones
- Full audio production (SFX + music)
- Shareable episode links
- Open web platform
- Faster generation (~60s vs 2-5 min)
